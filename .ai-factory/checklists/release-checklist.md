# Checklist de Release

## Pré-Release
- [ ] Todas as tarefas com V&V ✅
- [ ] INDEX.md reflete progresso real
- [ ] LOG-VALIDACOES.md atualizado
- [ ] Suite de testes backend passa
- [ ] Suite de testes frontend passa (se configurada)
- [ ] Nenhum bug blocker ou crítico aberto
- [ ] Performance report OK
- [ ] Security report OK
- [ ] Handoff completo (origem → devops)

## Release
- [ ] Branch main atualizada
- [ ] Tag de versão criada (v{major}.{minor}.{patch})
- [ ] CHANGELOG.md atualizado
- [ ] Deploy realizado
- [ ] Smoke test em produção OK

## Pós-Release
- [ ] Monitoramento ativo (1h pós-deploy)
- [ ] Erro report configurado
- [ ] Equipe notificada
