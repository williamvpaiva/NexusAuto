# NexusAuto — Executive Overview

## Propósito
Fábrica autônoma de software com 21 agentes de IA especializados que colaboram para produzir, validar e entregar software de qualidade corporativa.

## Stack
- **Frontend**: React 18 + TypeScript + Vite + TailwindCSS
- **Backend**: Node.js 20 + TypeScript + Express + Zod
- **Database**: SQLite (dev) / PostgreSQL (prod) + Prisma ORM
- **Infra**: Docker + Docker Compose

## Agentes
TECH-LEAD, CHIEF-OF-STAFF, PRODUCT-OWNER, ANALYST, ARCHITECT, FRONTEND-DEV, BACKEND-DEV, UI/UX-PRO-MAX, SECURITY, PERFORMANCE, QA-TESTER, DEVOPS, DATA-ANALYST, FINANCIAL-ANALYST, CUSTOMER-SUPPORT, PRODUCT-MARKETER, LEGAL-COUNSEL, RECRUITER, VENTURE-CAPITALIST, EXECUTOR, WHATSAPP

## Ferramentas Principais
- OpenWiki: Geração automática de documentação
- GNHF: Agentes overnight com V&V
- V&V Protocol: Gate de 7 passos pré-commit
- Memory Integration: Sync brain ↔ wiki
