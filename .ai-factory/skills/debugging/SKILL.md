# Superpoder
**Root cause analysis** sistemático - encontra a causa raiz em minutos, não horas.

# Trigger
- "debug this"
- "find the bug"
- "why is this failing"
- "troubleshoot"

# Método
1. **Reproduce** - Consistent reproduction
2. **Isolate** - Narrow down scope
3. **Analyze** - Logs, traces, metrics
4. **Fix** - Targeted solution
5. **Prevent** - Tests, monitoring

# Output
- Root cause identified
- Fix implemented
- Prevention measures
- Lessons learned
