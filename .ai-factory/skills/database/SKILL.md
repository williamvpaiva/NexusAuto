# Superpoder
Transforma queries **lentas** em **relâmpago** com tuning, índices e schema optimization.

# Trigger
- "slow query"
- "optimize database"
- "index strategy"
- "schema design"

# Técnicas
1. **Query Analysis** - EXPLAIN plans
2. **Index Strategy** - B-tree, composite, covering
3. **Schema Optimization** - Normalization vs denormalization
4. **Caching** - Redis, query cache

# Output
- Optimized queries
- Index recommendations
- Schema improvements
- Performance metrics (antes/depois)
