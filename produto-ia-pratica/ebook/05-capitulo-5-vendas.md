# Capítulo 5: IA para Vendas

## Venda Mais com Menos Esforço

Vendas é um jogo de números e relacionamento. IA não vai fechar negócio por você, mas vai acelerar cada etapa do processo: prospecção, abordagem, argumentação, follow-up e pós-venda.

---

## 1. Prospecção Inteligente

### Pesquisa de Leads

```
Quero prospectar clientes para [seu produto/serviço].

Me ajude a criar um perfil do lead ideal com:
- Características demográficas
- Dores que meu produto resolve
- Onde encontrar esses leads online
- Abordagem inicial recomendada

Depois, crie 3 mensagens de prospecção diferentes 
(LinkedIn, WhatsApp, Email) para esse perfil.
```

### Mensagem de Primeiro Contato

```
Crie uma mensagem de primeiro contato para [LinkedIn/WhatsApp/Email]
para oferecer [produto/serviço] para [perfil do cliente].

Regras:
- Máximo 2 parágrafos
- Não comece com "Olá, tudo bem?" (genérico demais)
- Primeira linha deve ser relevante para a pessoa
- Inclua uma pergunta para gerar resposta
- Sem links na primeira mensagem

Tom: consultivo, não vendedor.
```

---

## 2. Abordagem e Argumentação

### Script de Vendas (Estrutura SPIN)

```
Crie um script de vendas usando a metodologia SPIN para 
[produto/serviço]:

S - Situation (Situação): Perguntas para entender o cenário do cliente
P - Problem (Problema): Perguntas para identificar dores
I - Implication (Implicação): Perguntas para aumentar a dor
N - Need-payoff (Necessidade): Perguntas para mostrar a solução

Produto: [descreva]
Público: [descreva]
```

### Respondendo Objeções

```
Me dê respostas prontas para as 5 objeções mais comuns que recebo
em [seu negócio]:

1. "Está caro"
2. "Vou pensar"
3. "Preciso falar com meu sócio"
4. "Já tenho um fornecedor"
5. "Não é o momento"

Para cada objeção: 3 abordagens diferentes
(uma lógica, uma emocional, uma com prova social).
```

### Propostas Comerciais Personalizadas

```
Crie um template de proposta comercial personalizável para 
[seu serviço]. Estrutura:

1. Capa: Nome do cliente, sua empresa, data
2. Resumo executivo: 3 linhas sobre o que será feito
3. Diagnóstico: Entendimento da necessidade
4. Solução: O que será entregue
5. Cronograma: Prazos e marcos
6. Investimento: Opções (básico, completo, premium)
7. Garantia
8. Depoimentos
9. FAQ rápida
10. CTA e prazo da proposta
```

## 3. Follow-up e Fechamento

### Sequência de Follow-up

```
Crie uma sequência de 5 follow-ups para leads que pediram orçamento
mas não responderam. Intervalo: 3 dias entre cada.

Mídia: [WhatsApp/Email]

Sequência:
1. Follow-up 1: Reforçar valor (não "só lembrando")
2. Follow-up 2: Prova social (depoimento, caso)
3. Follow-up 3: Novidade ou bônus
4. Follow-up 4: Abordagem diferente (vídeo, áudio)
5. Follow-up 5: Última tentativa (rompimento)
```

### Técnicas de Fechamento

```
Me ajude a criar scripts de fechamento para [produto/serviço]
usando estas técnicas:

1. Fechamento Direto: "Vamos começar?"
2. Fechamento Presumptivo: "Prefere pagamento à vista ou parcelado?"
3. Fechamento por Escassez: "Últimas vagas/disponibilidade"
4. Fechamento por Alternativa: "Plano A ou Plano B?"
5. Fechamento por Benefício: "Qual benefício foi mais atrativo?"

Adapte cada um para meu negócio: [descreva]
```

## 4. Pós-Venda e Retenção

### Onboarding do Cliente

```
Crie um plano de onboarding de 7 dias para novos clientes de
[produto/serviço]. Objetivo: garantir que o cliente veja valor
rápido e não cancele.

Dia 1: Boas-vindas e acesso
Dia 2: Primeiro passo prático
Dia 3: Dica avançada
Dia 4: Pedido de feedback
Dia 5: Case de sucesso similar
Dia 6: Oferta exclusiva
Dia 7: Pesquisa de satisfação
```

### Pesquisa de Satisfação

```
Crie uma pesquisa de satisfação simples (5 perguntas) para enviar
para clientes que compraram [produto/serviço]. Inclua:

1 pergunta de NPS (0-10: "Recomendaria?")
2 perguntas quantitativas
2 perguntas abertas

A pesquisa deve ser rápida (menos de 2 minutos para responder).
```

## 5. CRM e Organização com IA

### Sugestão de Pipeline de Vendas

```
Projete um pipeline de vendas para [seu negócio] com 5-6 etapas.
Para cada etapa, defina:
- Nome da etapa
- Critério para entrar nela
- Ação principal do vendedor
- % de conversão esperada (estimativa)
- Como usar IA em cada etapa

Exemplo: Lead → Contato → Qualificação → Proposta → Negociação → Fechado
```

### Análise de Vendas

```
Analise meus dados de vendas do último mês:

[cole seus números: leads, conversões, ticket médio, etc.]

Me ajude a identificar:
1. O que está funcionando bem
2. O gargalo principal (onde mais perco?)
3. 3 ações para melhorar amanhã
4. Meta realista para o próximo mês
```

## Exercício Prático

Aplique agora: crie sua **sequência de follow-up de vendas**.

```
Prompt: "Crie uma sequência de 3 follow-ups para leads que 
receberam proposta mas não fecharam. Meu serviço: [descreva].
Valor médio: R$ [valor]. Tom: consultivo, não pressionador.
Inclua um gatilho de valor diferente em cada mensagem."
```
