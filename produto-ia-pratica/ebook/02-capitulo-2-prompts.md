# Capítulo 2: Prompts que Funcionam

## A Habilidade Mais Importante

Se a IA é sua assistente, o prompt é o briefing. Um bom prompt é a diferença entre uma resposta genérica e algo que você pode usar de verdade.

## A Fórmula de Ouro do Prompt

```
[CONTEXTO] + [TAREFA] + [FORMATO] + [TOM] + [EXEMPLO]
```

### 1. Contexto
Dê à IA o máximo de informações relevantes sobre a situação.

❌ "Escreva um post para Instagram."
✅ "Sou personal trainer, tenho 2 anos de experiência e quero um post para Instagram falando sobre a importância do aquecimento antes do treino. Meu público são iniciantes na academia."

### 2. Tarefa
Seja específico sobre o que você quer.

❌ "Me ajude com marketing."
✅ "Crie 5 legendas para Instagram sobre [tema] com até 150 caracteres cada, incluindo 3 hashtags relevantes."

### 3. Formato
Diga como quer a resposta.

❌ "Fale sobre vendas."
✅ "Crie um roteiro de call de vendas em formato de bullet points, com saudação, identificação de dor, apresentação de solução, objeções e fechamento."

### 4. Tom
Defina o estilo.

"Tom profissional" / "Tom casual e divertido" / "Tom autoritário" / "Tom de conversa" / "Tom educacional"

### 5. Exemplo (opcional, mas poderoso)
Mostre um exemplo do que você espera.

"Escreva algo no estilo deste post aqui: [cole o post]"

## 10 Prompts Coringa (Use e Adapte)

### Produtividade

```
1. Organizador de Tarefas
"Liste todas as tarefas que eu preciso fazer esta semana como
empreendedor na área de [sua área]. Organize em urgente x importante
e sugira uma rotina diária ideal."
```

```
2. Analisador de Decisões
"Preciso decidir entre [opção A] e [opção B]. 
Me ajude com uma análise de prós e contras de cada uma,
considerando meu contexto: [explique seu negócio]."
```

```
3. Resumidor
"Resuma o texto abaixo em 3 parágrafos, destacando 
os pontos mais importantes:
[cole o texto]"
```

### Marketing

```
4. Gerador de Ideias de Conteúdo
"Me dê 10 ideias de posts para Instagram sobre [tema].
Para cada ideia, inclua o título, a legenda principal e 3 hashtags."
```

```
5. Refinador de Texto
"Reescreva o texto abaixo para ser mais persuasivo e claro:
[cole seu texto]"
```

### Vendas

```
6. Respondedor de Objeções
"Sou vendedor de [seu produto/serviço]. Me ajude a responder
a seguinte objeção do cliente: '[cole a objeção]'.
Me dê 3 abordagens diferentes."
```

```
7. Seguidor de Lead
"Um cliente me procurou sobre [produto/serviço] há 3 dias
e não respondeu meu último contato. Escreva um follow-up
curto e educado por WhatsApp que gere uma resposta."
```

### Gestão

```
8. Pauta de Reunião
"Crie uma pauta para uma reunião de 30 minutos com minha equipe
sobre [tema]. Inclua objetivos, tópicos e tempo para cada um."
```

```
9. Simplificador de Processos
"Explique o processo de [algo complexo] em 5 passos simples.
Use analogias do dia a dia para facilitar o entendimento."
```

```
10. Brainstorming
"Faça um brainstorming comigo sobre [tema]. Me faça perguntas
para ajudar a gerar ideias melhores e depois compile as melhores opções."
```

## Técnicas Avançadas (Rápidas)

### Chain of Thought (Cadeia de Pensamento)
Peça para a IA "pensar passo a passo" antes de responder. Isso melhora drasticamente a qualidade.

❌ "Quanto devo cobrar pelo meu serviço?"
✅ "Pense passo a passo: quanto custa minha hora, qual o valor percebido pelo cliente, qual a média do mercado, qual minha margem ideal. Depois me dê um valor sugerido."

### Persona Fixa
Defina um personagem para a IA seguir durante toda a conversa.

"Você é meu consultor de marketing pessoal. Você tem 15 anos de experiência em marketing digital para pequenos negócios. Sempre responde de forma prática, direta e com exemplos reais."

### Prompt Chain (Corrente de Prompts)
Quebre uma tarefa complexa em prompts menores encadeados.

Prompt 1: "Me ajude a planejar o conteúdo do meu Instagram este mês."
Prompt 2 (após resposta): "Agora pegue a ideia 3 e desenvolva em um post completo."
Prompt 3: "Reescreva essa legenda em 3 versões diferentes."

## Erros Comuns ao Criar Prompts

| Erro | Solução |
|------|---------|
| Ser vago demais | Inclua contexto específico |
| Pedir muita coisa de uma vez | Quebre em prompts menores |
| Não definir formato | Especifique se quer lista, texto, tabela |
| Não revisar | IA erra. Você revisa. |
| Desistir no primeiro resultado | Refine. Peça ajustes. Teste. |
