# social-media-manager

## Purpose

Gerenciamento inteligente de conteúdo para redes sociais. Esta skill cria, agenda e otimiza posts para Instagram, LinkedIn, Twitter/X, Facebook e TikTok. Inclui análise de hashtags, otimização de horários de postagem, e geração de conteúdo multi-formato (carrossel, stories, reels, posts estáticos).

## When to Use This Skill

This skill should be used when:
- User needs to create social media content for multiple platforms
- User wants to schedule posts with optimal timing
- User needs hashtag research and optimization
- User wants to repurpose content across different platforms
- User needs engagement optimization strategies
- User wants to create content calendars
- User needs platform-specific content adaptation

## Core Capabilities

1. **Multi-Platform Content Creation** - Generate posts optimized for each social network
2. **Hashtag Strategy** - Research and suggest high-performing hashtags
3. **Content Calendar** - Plan and organize posting schedules
4. **Engagement Optimization** - Suggest best posting times and content types
5. **Content Repurposing** - Adapt single content piece for multiple platforms
6. **Analytics Integration** - Track performance metrics and suggest improvements

## Platform-Specific Guidelines

### Instagram
- **Feed Posts**: 2200 characters max, 3-5 hashtags in caption, 15-30 hashtags in first comment
- **Stories**: 500 characters max, interactive elements (polls, questions, quizzes)
- **Reels**: 60 seconds max, trending audio, text overlays, hooks in first 3 seconds
- **Carousels**: 10 slides max, storytelling flow, CTA on last slide
- **Best Times**: Tue-Thu 11am-1pm, engagement rate > 3%

### LinkedIn
- **Posts**: 3000 characters max, professional tone, 3-5 hashtags
- **Articles**: Long-form thought leadership, 800-2000 words
- **Best Times**: Tue-Thu 8am-10am, 12pm, 5-6pm
- **Content Types**: Industry insights, company updates, professional achievements

### Twitter/X
- **Tweets**: 280 characters, 1-2 hashtags, threads for longer content
- **Best Times**: Mon-Fri 8am-4pm, highest engagement Wed 12pm
- **Content Types**: Quick updates, thread storytelling, engagement questions

### Facebook
- **Posts**: 63206 characters max (use 40-80 for engagement), 1-3 hashtags
- **Best Times**: Wed-Fri 11am-1pm
- **Content Types**: Visual content, community building, events

### TikTok
- **Videos**: 10 minutes max (optimal 15-60 seconds), trending sounds
- **Best Times**: Tue-Thu 6am-10am, 7pm-11pm
- **Content Types**: Educational, entertainment, behind-the-scenes

## Content Creation Framework

### AIDA Model
1. **Attention** - Hook in first 3 seconds/words
2. **Interest** - Build curiosity and relevance
3. **Desire** - Show benefits and value
4. **Action** - Clear CTA (comment, share, click, follow)

### Content Pillars
- **Educational** (40%) - Tips, tutorials, how-tos
- **Engaging** (30%) - Questions, polls, stories
- **Inspirational** (20%) - Success stories, motivation
- **Promotional** (10%) - Products, services, offers

## Hashtag Strategy

### Research Process
1. **Competitor Analysis** - Top 5 competitors' performing hashtags
2. **Volume Check** - Mix of high (1M+), medium (100K-1M), low (<100K) volume
3. **Relevance Score** - Must relate to content and audience
4. **Performance Tracking** - Monitor which hashtags drive engagement

### Hashtag Sets
- **Branded** (2-3) - Company/product specific
- **Industry** (5-7) - Niche-specific terms
- **Trending** (2-3) - Current trending topics
- **Community** (3-5) - Community and movement tags

## Engagement Optimization

### Best Practices
- Respond to comments within 2 hours
- Ask questions to encourage interaction
- Use user-generated content
- Create shareable content (save-worthy)
- Post consistently (minimum 3x/week per platform)

### Metrics to Track
- **Engagement Rate**: (Likes + Comments + Shares) / Followers × 100
- **Reach**: Unique accounts that saw content
- **Impressions**: Total times content was displayed
- **Click-Through Rate**: Link clicks / Impressions × 100
- **Follower Growth**: Net new followers / period

## Workflow

### Content Creation Workflow
1. **Brief Collection** - Topic, goal, target audience, key message
2. **Platform Selection** - Which networks to publish on
3. **Content Adaptation** - Format for each platform
4. **Hashtag Research** - Platform-specific hashtag sets
5. **Visual Creation** - Images, videos, graphics
6. **Copy Writing** - Captions, CTAs, hooks
7. **Scheduling** - Optimal times for each platform
8. **Performance Tracking** - Monitor and optimize

### Crisis Management
- **Negative Comments**: Respond within 1 hour, take conversation private
- **PR Issues**: Pause scheduled posts, prepare holding statement
- **Algorithm Changes**: Diversify content types, increase engagement efforts

## Error Handling

### Platform API Issues
If social media APIs are unavailable:
```
⚠️  Unable to connect to [Platform] API

Options:
1. Create content manually (copy/paste format)
2. Retry connection in 5 minutes
3. Use alternative platform
4. Schedule for later when API is available
```

### Content Approval Required
If content needs review before posting:
```
📝 Content created and saved for review

Review checklist:
✓ Brand voice alignment
✓ Factual accuracy
✓ Legal compliance
✓ Visual quality
✓ Hashtag relevance

Ready for approval? [Y/n]
```

## Quality Standards

**Content Requirements:**
- Platform-optimized length and format
- Clear hook in first 3 seconds/words
- Single focused message per post
- Strong CTA aligned with goal
- Brand voice consistency
- Accessibility (alt text, captions)

**Hashtag Requirements:**
- 5-30 hashtags per platform (varies by network)
- Mix of volume levels
- Relevant to content and audience
- No banned or restricted tags
- Updated quarterly based on performance

**Visual Requirements:**
- High resolution (minimum 1080x1080px)
- Consistent brand colors and fonts
- Text overlay readable on mobile
- Alt text for accessibility
- Optimized file size for platform
