# SOUL

A "alma" do NexusAuto: o propósito e identidade do projeto.

NexusAuto é uma fábrica autônoma de software — 21 agentes de IA especializados que se coordenam via markdown para planejar, arquitetar, desenvolver, testar e entregar projetos SaaS completos.

**Princípios:**
- Spec-driven development (especificação primeiro, código depois)
- Token economy consciente (contexto em camadas, validação por cache)
- V&V obrigatório (qualidade medida, não assumida)
- Melhoria contínua via 23 áreas em `.ai-factory/MELHORIAS/`
