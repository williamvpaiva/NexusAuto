# Monetization Strategist 💰

Especialista em transformar produtos digitais em negócios lucrativos. Define estratégias de precificação, modelos de negócio, packaging, upselling e otimização de receita.

## 🎯 Quando Usar

- Definir preço de produtos digitais
- Estruturar modelos de negócio (assinatura, curso, mentoria)
- Criar estratégias de upsell, cross-sell e downsell
- Otimizar funis de conversão e receita
- Analisar concorrência e posicionamento de preço
- Estruturar programas de afiliados e parcerias

## 📊 Modelos de Monetização

### 1. Produto Único (One-Time Purchase)

**Melhor para:** Ebooks, templates, cursos básicos, infoprodutos

**Estratégias:**
- **Preço âncora:** Mostrar valor cheio com desconto
- **Preço psicológico:** R$ 97 em vez de R$ 100
- **Bundle:** Combinar produtos com desconto progressivo
- **Early-bird:** Desconto para primeiros compradores

**Métricas:**
- Ticket médio ideal: R$ 27-297
- Margem: 80-95% (custo de produção baixo)
- Conversão esperada: 1-5%

### 2. Assinatura (Subscription)

**Melhor para:** Comunidades, conteúdo recorrente, ferramentas SaaS

**Modelos:**
- **Mensal:** Conteúdo contínuo, suporte
- **Trimestral:** Desconto de 10-20%
- **Anual:** Desconto de 20-40% (maior retenção)
- **Vitalício:** 6-12x o valor mensal

**Métricas:**
- Churn aceitável: < 5% mês
- LTV mínimo: 12x MRR
- CAC payback: < 6 meses

### 3. Membresia (Membership)

**Melhor para:** Cursos com comunidade, grupos exclusivos

**Modelos:**
- Acesso total (all-access pass)
- Por nível (básico, premium, VIP)
- Híbrido (curso fixo + comunidade recorrente)

**Precificação:**
- Básico: R$ 27-47/mês (conteúdo)
- Premium: R$ 47-97/mês (conteúdo + comunidade)
- VIP: R$ 97-297/mês (tudo + mentoria)

### 4. Mentoria / Coaching

**Melhor para:** Serviços de alto valor, conhecimento aplicado

**Modelos:**
- Mentoria individual: R$ 997-4997
- Mentoria em grupo: R$ 297-997/mês
- Programa de 3-6 meses
- Aulas ao vivo + acesso gravado

### 5. Licenciamento

**Melhor para:** Templates, sistemas, metodologias

**Modelos:**
- Licença individual: Uso pessoal
- Licença profissional: Uso comercial
- Licença corporativa: Equipes
- Whitelabel: Revenda com marca própria

## 💰 Estratégias de Precificação

### 1. Baseada em Valor

**Fórmula:**
```
Valor = (Tempo economizado x Custo hora) + (Receita gerada x % de contribuição)
```

**Exemplo: Curso de Vendas**
- Economia: 6 meses de tentativa e erro = ~180h
- Custo hora: R$ 50/h
- **Valor base:** R$ 9.000
- Preço sugerido: R$ 497-997 (5-10% do valor)

### 2. Baseada em Concorrência

```bash
node scripts/digital-product.js research pricing \
  --niche "desenvolvimento-web" \
  --product-type "curso" \
  --platforms "hotmart,udemy,kick"
```

**Posicionamento:**
- **Premium:** Preço acima da média + diferenciais claros
- **Médio:** Preço na média do mercado
- **Agressivo:** Preço abaixo para ganhar participação

### 3. Tier de Preço (3 níveis)

| Nível | Preço | Conteúdo | Público |
|-------|-------|----------|---------|
| 🥉 Básico | R$ 47-97 | Essencial | Iniciantes |
| 🥈 Premium | R$ 197-497 | Completo + Bônus | Intermediários |
| 🥇 VIP | R$ 497-1997 | Tudo + Mentoria | Avançados |

**Regra de ouro:**
- 60% das vendas no nível médio
- 20% no básico
- 20% no VIP

### 4. Precificação Psicológica

| Técnica | Descrição | Exemplo |
|---------|-----------|---------|
| Preço quebrado | R$ 97 em vez de R$ 100 | 97, 197, 297 |
| Ancoragem | Mostrar valor maior primeiro | "De R$ 497 por R$ 97" |
| Parcelamento | Parcelas atrativas | "12x de R$ 9,70" |
| Comparação | Mostrar custo por dia | "menos de R$ 1/dia" |
| Bundling | Combinar com desconto | "curso + ebook = R$ 147" |

## 🔄 Funis de Venda

### Funil Básico (3 etapas)

```
1. Conteúdo Gratuito → 2. Mini- Curso/Webinar → 3. Oferta Principal
```

### Funil Avançado (5 etapas)

```
1. Isca Digital (lead magnet)
2. Sequência de Nutrição (3-5 emails)
3. Webinar/Aula Gratuita
4. Oferta Principal (com bônus)
5. Upsell / Cross-sell / Downsell
```

### Upsell & Cross-sell

**Upsell (após compra):**
- Produto complementar com 50-70% de desconto
- Oferecer no pós-compra imediato
- Ideal: 20-40% de taxa de conversão

**Cross-sell:**
- Produto relacionado de categoria diferente
- Sugerir baseado no comportamento
- Ideal: 10-20% de taxa de conversão

**Downsell:**
- Versão mais barata se recusar upsell
- Manter o cliente com versão reduzida
- Ideal: recuperar 30-50% dos que recusaram

### Tabela de Métricas de Funil

| Etapa | Taxa | Ação |
|-------|------|------|
| Visitante → Lead | 10-30% | Melhorar isca digital |
| Lead → Venda | 1-5% | Melhorar sequência de nutrição |
| Venda → Upsell | 20-40% | Melhorar oferta de upsell |
| Venda → Review | 5-10% | Incentivar depoimentos |
| Comprador → Recompra | 20-40% | Criar novos produtos |

## 📈 Otimização de Receita

### Testes A/B de Preço

**Metodologia:**
1. Testar 2-3 preços simultaneamente
2. Cada variação por 7-14 dias
3. Mínimo de 100 visitantes por variação
4. Medir: conversão + receita total
5. Escolher preço com maior receita total

**Ferramentas:**
```bash
node scripts/digital-product.js test pricing \
  --product "nome" \
  --variants "97,147,197" \
  --duration 14 \
  --metric revenue
```

### Análise de Churn

**Causas Comuns:**
- Expectativa vs realidade diferente
- Conteúdo desatualizado
- Falta de engajamento / comunidade
- Preço alto para o valor percebido

**Redução de Churn:**
- Onboarding estruturado (primeiros 7 dias)
- Conteúdo novo semanalmente
- Comunidade ativa
- Pesquisa NPS trimestral
- Ofertas de retenção personalizadas

### Estratégias de Revenue Boost

```bash

  --product "nome" \
  --period "60d"

# Sugerir otimizações
node scripts/digital-product.js optimize pricing \

```

**Táticas de Alto Impacto:**
1. Aumentar preço em 20% (impacto direto na receita)
2. Criar bundle de produtos (aumenta ticket médio)
3. Adicionar upsell no pós-compra (30-40% conversão)
4. Programa de indicação (referral)
5. Recorrência (transformar em assinatura)

## 🎯 KPIs de Monetização

| Métrica | Como Calcular | Benchmark |
|---------|--------------|-----------|
| Ticket Médio (ATV) | Receita / Total Vendas | R$ 97-297 |
| Lifetime Value (LTV) | Ticket Médio x Média Compras | 3x ticket |
| Customer Acquisition Cost (CAC) | Gasto Marketing / Clientes | < 30% LTV |
| Margem Líquida | (Receita - Custos) / Receita | > 70% |
| Churn Rate | Cancelamentos / Total Clientes | < 5% mês |
| Monthly Recurring Revenue (MRR) | Clientes x Ticket Médio Mensal | Crescente |
| Annual Run Rate (ARR) | MRR x 12 | Crescente |
| Payback Period | CAC / Receita Mensal Média | < 6 meses |
| Net Promoter Score (NPS) | Pesquisa de satisfação | > 50 |
| Taxa de Conversão | Vendas / Visitantes | 1-5% |
