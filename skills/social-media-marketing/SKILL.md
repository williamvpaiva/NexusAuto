## Platform Guides

### Twitter/X
```markdown
## Post Structure
- Hook: First 280 chars must grab attention
- Value: Deliver insight/idea
- CTA: Engage with question or share

## Thread Format
1. Hook tweet (tease the thread)
2. Thread body (5-10 tweets max)
3. CTA tweet (follow, engage, link)

## Best Practices
- Use line breaks for readability
- Include relevant hashtags (1-2)
- Add media (image/video) when possible
```

### Instagram
```markdown
## Post Types
- Carousel: Educational content (5-10 slides)
- Static: Single impactful image + caption
- Reels: Short-form video (15-60s)
- Stories: Ephemeral, casual content

## Caption Structure
1. Hook: First line must stop scroll
2. Body: Value + story
3. CTA: "Save this post", "Share", "Comment"

## Hashtag Strategy
- 20-30 hashtags per post
- Mix: 10 popular + 10 niche + 5 specific
```

### LinkedIn
```markdown
## Post Structure
- Professional insight or story
- 3-5 paragraphs max
- Include relevant experience
- Thought-provoking question or CTA

## Content Types
- Industry insights
- Career lessons
- Company culture
- Professional achievements
- How-to threads
```

## Engagement Analytics

### Key Metrics
| Metric | What It Measures | Target |
|--------|------------------|--------|
| Reach | How many saw content | Platform dependent |
| Engagement Rate | Likes + Comments / Reach | 3-5% |
| Click-through | Link clicks | 1-3% |
| Saves | Content value | High = great |

### Performance Review
```markdown
# Weekly Review
- [ ] Which posts performed best?
- [ ] What content drove engagement?
- [ ] What timing worked?
- [ ] What to repeat next week?
```
